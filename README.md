# skills-workflows

Pipeline orchestration and workflow skills for Claude Code. 8 skills total: 5 `workflow-*` orchestrators plus `publish-post`, `agents-md-generator`, and `work-tracking`.

Install via the [slogsdon-claude-code-config marketplace](https://github.com/slogsdon/claude-code-config).

## Skills

- **publish-post** — Publishes content through the post pipeline
- **agents-md-generator** — Generates AGENTS.md for multi-language sample repos
- **work-tracking** — Markdown-based task tracking (TASKS.md)
- **workflow-design** — Design workflow pipeline orchestrator
- **workflow-feature-spec** — Feature specification workflow
- **workflow-stage-draft** — Stage draft workflow
- **workflow-vault-weekly** — Vault weekly workflow
- **workflow-monthly-skill-audit** — Monthly skill audit workflow

## Reference

- **_workflow-lib/** — Shared workflow library (not a skill; used by the workflow skills above)
